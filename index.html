End-to-End dbt & Databricks Data Engineering Project | Modern Analytics Engineering Pipeline

Developed an end-to-end analytics engineering pipeline using dbt Core and Databricks, implementing the Medallion Architecture (Bronze, Silver, Gold) to transform raw data into clean, analytics-ready models. The project involved ingesting multiple datasets into Databricks, performing modular SQL transformations with Jinja, building reusable dbt macros, implementing data quality tests, seeds, and snapshots for historical tracking, and managing the complete development lifecycle using Git, GitHub, Visual Studio Code, Python, and uv. The project was structured following modern data engineering best practices and prepared for CI/CD deployment.
